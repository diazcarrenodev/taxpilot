# TaxPilot

TaxPilot is an MVP web app to help organize inputs for Colombian personal income tax (Renta PN)
and generate a clear summary to reduce errors and time.

## MVP Scope
- Manual input form (no DIAN integration)
- Basic validation + checklist of required docs
- Tax summary report (PDF later)

## Roadmap
- v0.1: checklist + manual inputs
- v0.2: PDF export
- v0.3: document parsing (PDF/OCR) experiments

## Tech (planned)
- Next.js (web)
- Node.js (api)
- PostgreSQL (db)
